# 快速开始指南

## 🚀 5分钟上手

### 第一步：安装依赖

首次运行会自动安装，或手动执行：

```bash
# 创建虚拟环境
python3 -m venv venv

# 激活虚拟环境
# macOS/Linux:
source venv/bin/activate
# Windows:
venv\Scripts\activate

# 安装依赖
pip install -r requirements.txt
```

### 第二步：初始化数据库

```bash
python -c "from utils.db_helper import init_db; init_db()"
```

### 第三步：启动应用

```bash
# macOS/Linux
./start.sh

# Windows
start.bat

# 或直接运行
streamlit run app.py
```

访问：**http://localhost:8501**

## 📖 使用流程

### 1️⃣ 添加股票

在"数据管理" → "股票池管理"中：
- 添加股票代码（格式：600519.SH）
- 或批量导入（每行一个代码）

### 2️⃣ 下载数据

在"数据管理" → "数据下载"中：
- 选择要下载的股票
- 选择日期范围
- 点击"开始下载"

### 3️⃣ 计算指标

在"技术指标" → "指标计算"中：
- 选择股票
- 点击"计算指标"
- 等待计算完成

### 4️⃣ 配置策略

在"策略配置" → "配置策略"中：
- 选择策略类型（趋势/突破/震荡）
- 调整参数
- 保存策略

### 5️⃣ 执行选股

在"执行选股"中：
- 选择已保存的策略
- 选择股票池
- 点击"开始选股"
- 查看结果

## 💡 常用操作

### 股票代码格式

- 上海市场：600519.SH（贵州茅台）
- 深圳市场：000001.SZ（平安银行）

### 数据周期

- d：日线
- 60m：60分钟
- 30m：30分钟
- 15m：15分钟
- 5m：5分钟

### 策略类型

1. **趋势型**：适合牛市
   - 均线多头排列
   - MACD金叉
   - 回踩均线
   - 趋势跟踪

2. **突破型**：适合震荡市
   - 箱体突破
   - 放量突破
   - 连续上涨
   - 布林带突破

3. **震荡型**：适合熊市
   - KDJ低位金叉
   - RSI超卖
   - 双超卖
   - 布林带下轨

## ⚠️ 注意事项

1. **首次使用**：需要先添加股票并下载历史数据
2. **数据限制**：baostock有请求频率限制，大量下载请使用延迟
3. **数据完整性**：确保数据至少30条才能计算所有指标
4. **策略选择**：根据市场环境选择合适的策略类型
5. **风险控制**：不要全仓一只股票，设置止损

## 🔧 常见问题

### Q: 提示"stock_data.db不存在"？
A: 运行 `python -c "from utils.db_helper import init_db; init_db()"` 初始化数据库

### Q: 下载失败？
A:
- 检查网络连接
- 确认股票代码格式正确
- 降低请求频率

### Q: 指标计算失败？
A:
- 确保数据至少30条
- 检查数据是否完整
- 重新下载股票数据

### Q: 选股没有结果？
A:
- 检查策略参数是否过于严格
- 尝试放宽条件
- 确认已计算指标
